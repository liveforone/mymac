# Mac Command

## mise
- `mise current`
- `mise upgrade`
- `mise self-update` : mise version update

## brew
- `brew update && upgrade`
- `brew cleanup —prune=all`
- `brew list`

## ghostty
- `Cmd + t` : new tap
- `Cmd + w` : close tap
- `Cmd + d` : Vertical split
- `Cmd + shift + d` : Horizontal split
- `Cmd + opt + arrowKey` : move pane
- `Cmd + backspace` : erase all command line

## nvim & plugins
- `space + e` : Neotree toggle
- `Control + w + h/l` : move Neotree and editor
    - a : new file/dir
    - d : delete file/dir
    - r : rename file/dir
    - y : yanking file/dir
    - x : cut file/dir
    - p : paste file/dir
    - m : move file/dir
- `space + ff` : fuzzy finding
- `space + fg` : live grep
- `:Lazy sync` : Install
- `:Lasy update` : Update
- `:sp` : split
- `:vsp` : vertical split

## zsh & git shortcut command

- If you want to see zsh or git shortcut command,
- look `.config/dotfiles/aliases.zsh`
- If you don’t like it, you can change the commands however you want.

## Mac os command - KOR

- `Cmd + backspace` : 휴지통 보내기
- `Cmd + opt + backspace` : 완전히 삭제
- `Cmd + space` :  spotlight
- `Opt + space` : 런치패드
- `Opt + cmd + space` : finder
- `Opt : d` : 바탕화면 바로가기
- `click + space` : 미리보기
- `Ctrl + shift + 3` : 스크린샷
- `F3(우측 opt)` : mission control
- `Cmd + opt + esc` : 앱 강제종료 창
- `Cmd 방향키` : home end
- `Fn 방향키` : pgup pgdown

## Mac os command - ENG

- `Cmd + Backspace` : Move to Trash
- `Cmd + Option + Backspace` : Delete Immediately
- `Cmd + Space` : Spotlight
- `Option + Space` : Launchpad
- `Option + Cmd + Space` : Open Finder
- `Option + D` : Go to Desktop
- `Click + Space` : Quick Look Preview
- `Ctrl + Shift + 3` : Screenshot
- `F3 (right Option)` : Mission Control
- `Cmd + Option + Esc` : Force Quit Applications
- `Cmd + Arrow Keys` : Home / End
- `Fn + Arrow Keys` : Page Up / Page Down
